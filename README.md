# RaceSync

Mobile-first sim racing schedule and practice planner for iOS and Android.

## Overview

RaceSync helps sim racers answer two fundamental questions:
1. **When are races happening?** View upcoming race schedules across multiple sim racing series
2. **What should I practice?** Plan practice sessions for the right conditions (track, car, time of day)

Built with React Native for true cross-platform mobile experience, designed for one-handed use while at your sim rig.

## Key Features (Planned)

- Offline-first race schedule viewing
- Calendar integration for race reminders
- Practice session planning
- Multi-sim support (iRacing, ACC, and more)
- Privacy-focused: your data stays on your device
